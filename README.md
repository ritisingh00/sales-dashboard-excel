# 📊 Sales Analytics Dashboard — Microsoft Excel

An interactive sales analytics dashboard built using advanced Microsoft Excel features to visualize and analyze sales performance data.

---

## 📌 Overview

This project demonstrates end-to-end data analysis and visualization using Microsoft Excel — from raw data cleaning to an interactive dashboard with dynamic filters.

---

## ✨ Features

- 📈 **Interactive Dashboard** with dynamic charts and KPI cards
- 🔍 **Slicers & Timelines** for filtering by region, product, and time period
- 🔄 **Power Query** for data cleaning and transformation
- 📐 **Pivot Tables** for multi-dimensional data summarization
- 🎨 **Conditional Formatting** to highlight trends and outliers
- ✅ **Data Validation** for clean data entry
- 🧮 **Nested IF Formulas** for dynamic categorization
- ⚡ **Macros / VBA** to automate repetitive data refresh tasks

---

## 📊 Dashboard Highlights

| KPI | Description |
|-----|-------------|
| Total Revenue | Overall sales revenue across all regions |
| Top Products | Best performing products by sales volume |
| Regional Breakdown | Sales distribution across regions |
| Monthly Trend | Revenue trend line over time |
| Growth % | Month-over-month growth comparison |

---

## 🚀 How to Use

1. Download `sales_dashboard.xlsx`
2. Open in Microsoft Excel (2016 or later recommended)
3. Enable Macros if prompted
4. Use the **Slicers** on the dashboard to filter by Region, Product Category, or Time Period
5. Data refreshes automatically via Power Query connection
